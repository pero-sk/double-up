# Double Up Joker

A Balatro mod that adds a new joker, a tarot card, and an unlockable deck. All favouring a Two Pair.

## Features

- Double Up Joker (Uncommon)
  - Grants +12 Mult if the played hand is Two Pair and any cards share the same suit.
- The Mirror (Tarot)
  - Select 3 cards. The 2 left-most cards are converted into the right-most card. This copies rank and suit only.
- Suited Mirror (Deck)
  - Start the run with The Mirror and Double Up Joker.

## Unlock Condition

Suited Mirror is unlocked by using The Mirror and triggering Double Up Joker in the same run.

## Localisation

The mod includes:
- English (en-us)
- Polish (pl)

## Installation

1. Install Steamodded
2. Download or clone this repository
3. Place the mod folder into:
```
AppData\Roaming\Balatro\Mods\
```
4. Launch the game

## Requirements

- Balatro
- Steamodded
